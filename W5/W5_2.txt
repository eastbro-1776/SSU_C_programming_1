다음 프로그램을 실행해보아라.

#include <stdio.h>
int main(void)
{
   int num;

   printf("숫자를 입력하세요 : "); 
   scanf("%d", &num);
   printf("%d의 거듭제곱은 %d 입니다.\n", num, num * num);
   return 0;
}

• 7을 입력했을 때의 결과 : 7의 거듭제곱은 49입니다.

• 100을 입력했을 때의 결과 : 100의 거듭제곱은 10000입니다.

• 10.23을 입력했을 때의 결과 : 10의 거듭제곱은 100입니다.

• 10abc를 입력했을 때의 결과 : 10의 거듭제곱은 100입니다.

• a를 입력한 결과 :  0의 거듭제곱은 0입니다.

• 위의 결과를 보고 scanf() 함수가 정수를 어떻게 입력받는지 설명하여라.
10.23의 경우 1, 0, ., 2, 3 으로 입력되는데 .은 문자로 인식된다. 10까지 입력을 받다가 .으로 인해 scanf가 종료된다. 따라서 입력받은 숫자는 10.
